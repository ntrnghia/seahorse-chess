    Quick Rules

    • Goal: Get all 4 horses into the deepest home steps (3, 4, 5, 6).
    • Exit: Roll doubles or (1,6) to release a horse onto your exit cell. Bonus turn granted on every doubles or (1,6).
    • Move: Each turn move one horse by 0, X, Y, or X+Y steps. Track loops counter-clockwise.
    • Cards: Each exiting horse draws a hidden card. Only the owner sees it (your own cards appear as text on the horse).
    • Combat: Higher card wins. Loser returns to stable and discards card publicly; defender loses 1 life. 0 lives → stable & lose card.
    • Combat visibility: Both fighters reveal cards to each other. Third parties only see the loser's card (via the public discard).
    • Specials: Joker draws 3 cards, highest counts. Soul Stealer wins by stealing the rival's card identity.
    • Home entry: Free entry from the main track. Inside the home stretch, a single die must exactly match your next step (or use both dice for the rare two-step jump).
    • Free Exit: 3 failed exits in a row with no horse on the track → next turn one free exit is granted.
    • PVP: Create a room, share the 9-digit code or invite link. Add bots in empty slots. Host clicks Start when all 4 slots are filled.
